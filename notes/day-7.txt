Functional Programming:
-----------------------------------
Functional programming (FP) is a declarative paradigm that constructs programs by composing pure, side-effect-free functions while avoiding mutable state and shared data. 
It emphasizes immutability, treating data as constant, and uses first-class functions to create modular, predictable code.

Pure function:
--------------------
a function which does not mutate (change) the value of the inputs and simply produces a result out of those inputs

20. Extension Method
this technique is used to "extend" (not by inheritance) an exsting class or interface by "injecting" new methods without modifying the source code of that class or interface

technique:
create a separate class
mark the class with 'static' keyword (means all the method here will be static method)
create the extra methods (that you want to add in an existing class or interface) in the following manner

public static class CalculatioExtension{
	public static int Multiply(this Calculation calc,int a, int b){
		int res  = calc.Add(a,b);
		return a-b-res;
	}
}

you will find that the extra method has been included in that existing class/interface

Note: ONLY INSTANCE METHODS CAN BE ADDED IN THIS WAY.NO STATIC METHOD CAN'T BE ADDD USIN EXTENSION METHOD TECNIQUE.

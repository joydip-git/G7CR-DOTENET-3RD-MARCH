Functional Programming:
-----------------------------------
Functional programming (FP) is a declarative paradigm that constructs programs by composing pure, side-effect-free functions while avoiding mutable state and shared data. 
It emphasizes immutability, treating data as constant, and uses first-class functions to create modular, predictable code.

Pure function:
--------------------
a function which does not mutate (change) the value of the inputs and simply produces a result out of those inputs

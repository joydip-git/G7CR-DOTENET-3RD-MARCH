if in a class you have NOT written any constructor explicitly [neither default nor parameterized], then default constructor will be supplied

class Sample{}

if in a class you have written a parameterized constructor, then default constructor will NOT be supplied

class Sample
{
	string data;
	public Sample(string data) => this.data = data;
}

primary constructor:
	if the class has one and only one parameterized constructor and a series of data members that are assigned by that constructor, in that case you write the above code as the following:

class Sample(string data)

{

	int y;

	public Some(){}

	public Some(int x) => this.x = x;

	public Some(int x, int y) {this.x = x; this.y = y;}

constructor execution sequence:
-----------------------------------
during the creation of a child class object, constructors from both parent and child gets invoked
	- by default, default constructor of base class is executed first
	- then the constructor of the child class (whichever you called) is executed

constructor execution sequence: calling base class's constructor explicitly
-----------------------------------------------------------------------------
you can explicitly invoke a constructor of a base class from the constructor fo a child class by using "base" keyword
